# Addons

This page contains **unofficial** addons that can be installed on top of Apostasy. Support for these addons is provided by the creator of the addon in the `#Apostasy-modifications` channel in the [Waking Dreams](https://discord.gg/4WwqfK5yHg) discord server. Please do not seek support for these addons in the official support channels.  

If you would like to make an addon and for it to be featured on this page, please reach out to me (aljo) in the modifications channel and we can discuss it.
